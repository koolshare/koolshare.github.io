小宝开源插件
=========

目前仅支持小宝改版本的 Merlin 固件的插件。未来有望支持 Tomato/Ddwrt 等 arm 系列路由器的插件而成为类似 Entware 可以让大家安装并且有配置界面的插件列表。目前还远远达不到我们心中的设计目标，时间有限，只能一跳一跳的去完成，希望能有更多兄弟参与。

* 讯雷下载
* Aria2 下载
* 讯雷快鸟

* WebShell
* 远程消息通知
* Alidns + ddnspod + gddns
